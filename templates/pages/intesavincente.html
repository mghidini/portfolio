Intesa Vincente

View on GitHub

Overview:

Intesa Vincente is an Android Application that draws inspiration from the popular italian TV quiz show “Reazione a Catena” and offers the opportunity to play the game "L’Intesa Vincente", featured in the show.

The game involves a team of three players: two players, the clue-givers, are presented with the same randomly selected word.
By each saying only one word at a time, they must formulate coherent sentences to describe the word and help the third player, the guesser, figure it out.

The application is designed to support the real game, offering interfaces that engage users and manage game data to make the gameplay fun and dynamic.

My Role:

The Intesa Vincente app was designed and developed for the Mobile Development Programming course, at Università degli Studi Milano Bicocca, in a 4 members team.

Technologies Used:

The application is developed using Java, the Gradle build system, and the Android Studio IDE.
Other external services have been used, such as Firebase and the Retrofit library.

Key Features:

Login

The user can log into the application. There are two login options available: via email and password, and via Google Account.

Registration

The user registers for the application by entering their email address, password, and a chosen nickname.

Logout

The user can log out of the application.

Edit Profile

The user can modify their personal information, specifically email, password, and nickname.

Play

The user can start a game by choosing from various modes and options that best suit their preferences (creating a new group or joining an existing one, selecting the role to play, etc.).
The simulated game requires three players and thus three devices.
Two players (Clue-Givers) will be shown the same randomly selected word on their devices to make the third player guess, according to the game's rules.
On the Guesser's device, guessed words, available passes, and remaining time are displayed.
The Guesser can manage the game via buttons to pause and resume time, use passes, and enter guessed words.

Statistics

The user can view all the information related to the games they have played over time, including the number of games played, won and lost, the highest number of points obtained in a single game and the total number of words guessed.

Friends

The user can view the list of users they have played with.

Solutions Implemented:

The app was designed according to Google Android guidelines and Material Design principles.

Results:

The final project was evaluated by the examination board and received a score of 29/30.
