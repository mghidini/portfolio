Dardos

View on GitHub

Overview:

The project Dardos implements a method to Identify and score darts thrown into dartboard, using segmentation and image processing tools.

My Role:

This project was completed as part of the coursework for Digital Image Processing, at Universidad de Sevilla, in a 2 members team.

Technologies Used:

The work has been carried out using MATLAB.

Academic Research:

The work was supported by academic research in the following fields:

  • Image segmentation techniques.
  • The Hough transform technique for extraction in digital image processing.
  • Canny algorithm for edge detection.

Objectives:

The project involves locating a dart on a dartboard in an image to determine the score.
We have a classic circular dartboard divided into twenty sections, alternating colors between white and black, with two concentric circles (bullseye).
There are also two outer rings, alternating in green and red, which further divide the dartboard into double and triple score zones.
To accurately calculate the points, we must correctly determine the location of the dart on the dartboard and the corresponding points for that location. This is achieved using digital image processing techniques.

Solutions Implemented:

The problem is divided into two parts.
In the first part, we will segment the dartboard image to assign the corresponding points to each pixel.
The second task is to locate the dart, determining the pixels in the image that correspond to the point where the dart is in contact with the dartboard.

Dartboard Segmentation

To obtain the dartboar segmentation, we use image processing functions and techniques.
We first separated the dartboard from the image backgroung; then we segmented the multiplication zones (x2, x3, bullseye, and inner bullseye) by color, specifically green and red.
Finally, we obtained the radial division of the dartboard and assigned the correct points to all scoring zones.

Dart Identification

To identify the dart on the dartboard, we implemented an algorithm that extracts the object based on the RGB channels, specifically the blue channel.
Once the region of the dart is obtained, the next step is to find the endpoints, specifically the point where the dart tip is located, to correctly assign the points.
To determine which end is the dart tip, we use the orientation of the dart in the image.

Scoring Algorithm

Once the division of the dartboard is obtained, each region is assigned its corresponding score.
The coordinates of the pixel where the dart tip is located are then taken to calculate the obtained points.
To verify if the score is single, double, triple, or null, we use the masks previously created with the dartboard segmentation algorithm.

Implementation:

The project is developed in MATLAB and consists of a main script, Dardos.m, and two functions, segmentarDiana.m and getScore.m.
When the script Dardos.m is executed, the program prompts the user to load the photo of the dartboard and the photo of the dartboard with the dart.
Then, the function segmentarDiana.m is called to divide the dartboard into different regions.
Finally the function getScore.m is called and the correct score is presented to the user.
